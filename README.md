# Insurance Claims Dashboard (Tableau)

## 📊 Project Overview
This project demonstrates an interactive Tableau dashboard built using an insurance dataset.  
The goal is to analyze customer demographics, policy details, and claim behavior to uncover insights that can support business decision-making.

## 🗂 Dataset
The dataset includes customer and policy-related attributes such as:
- **Customer Information**: State, Gender, Marital Status, Education, Employment Status, Income  
- **Policy Details**: Coverage, Policy Type, Policy Level, Renew Offer Type, Sales Channel  
- **Insurance Metrics**: Customer Lifetime Value, Monthly Premium Auto, Total Claim Amount  
- **Vehicle Information**: Vehicle Class, Vehicle Size  
- **Other Attributes**: Months Since Last Claim, Number of Policies, Number of Open Complaints

## 🛠 Features
- Interactive filters for **State, Coverage, Policy Type, and Vehicle Class**
- KPIs such as **Customer Lifetime Value, Monthly Premiums, and Claim Amounts**
- Trend analysis of **claims vs. policies**
- Segmentation by **customer demographics and vehicle size**
- Visual storytelling with charts, maps, and dashboards

## 🎯 Insights
- Identify high-value customers based on **Customer Lifetime Value (CLV)**
- Compare claim amounts across different **policy types and coverage levels**
- Spot trends in **customer demographics** influencing insurance claims
- Evaluate **sales channels** and their impact on renewals

## 🧰 Tools & Skills Demonstrated
- **Tableau**: Dashboard creation, calculated fields, parameters, interactive filters
- **Data Analysis**: Trend identification, KPI reporting, segmentation
- **Business Intelligence**: Data storytelling, visualization for decision-making
